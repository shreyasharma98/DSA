## USER DEFINED FUNCTIONS

# 1.oneToN
Given a number n, print 1 to n. 
<br>
 ### Sample Input :<br>
```
8 
```


### Sample Output :<br>
``` 
1 2 3 4 5 6 7 8
```

# 2.Fibonacci Number

Given a number N, figure out if it is a member of fibonacci series or not. Return true if the number is member of fibonacci series else false.
 ### Sample Input :<br>

```
5
```
### Sample Output :<br>
``` 
true
```
